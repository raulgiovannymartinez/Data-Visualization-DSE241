--> How to run application?

1. Make sure the data files below are available:
	./data/data.pkl
	./data/coordinates.pkl

2. Create a local environment with Python version 3.7.9

3. Install requirements as listed in the requirements.txt file.
   Use the following command to install all requirements:
	
	pip install -r requirements.txt
	

4. Run python file 'app.py'

5. Open browser and enter localhost address to see application: 
	http://127.0.0.1:8050/
